🏏 Cricket Master – README

📌 Overview
Cricket Master is a cricket prediction and fantasy app built using Python + Streamlit + Machine Learning.
It provides live match data, AI predictions, fantasy gameplay, and offline support.

✨ Features
📡 Live Match Scorecard (real-time updates)
🔮 AI Match Predictions with probability & confidence
🎮 Fantasy Cricket Game (₹100 Cr budget system)
📊 Analytics Dashboard (model insights)
📰 Cricket News (latest updates)
🌐 Offline Support (PWA enabled)
🔔 Smart Notifications


🧠 Machine Learning
Model: Ensemble (Random Forest + Gradient Boosting)
Dataset: 2000+ matches
Accuracy: 75.8%
Key Factors:
Team strength
Recent form
Venue advantage
Toss impact
Head-to-head stats


🛠 Tech Stack
Frontend: Streamlit
Backend: Python
ML: Scikit-learn
Database: SQLite
APIs: Cricket API, ESPN data


▶️ Run the Project
python data_fetcher.py
python advanced_model.py
streamlit run app.py

Open in browser:

http://localhost:8501
